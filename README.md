## About the Project

This repository is dedicated to helping 42 Network students prepare for the **new exam formats (2025/2026 update)**. As the curriculum evolves, so do the exam pools.  
 The goal here is to provide clean, reliable, and deeply explained solutions to help you understand the core concepts behind the questions rather than just rote-memorization.

---

## Current Content Roadmap

### Rank 03 
Focuses on advanced string manipulation, file descriptors, memory allocation, and basic algorithms.
*   **Key Concepts:** `ft_scanf` concepts, custom `get_next_line` implementations, and initial backtracking.
*   **Structure:** `/rank_3/` contains subject files and solutions.

### Rank 04 
Tackles complex algorithms, system calls, and fundamental UNIX/Linux operations.
*   **Key Concepts:**  Advanced backtracking, and recursive algorithms.
*   **Structure:** `/rank_4/` contains subject files and solutions.
